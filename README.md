# MERN Personal Task Manager

This is a full-stack Personal Task Manager application built using the MERN stack.

## Features
- User Registration & Login (JWT Authentication)
- Add, Edit, Delete Tasks
- User-specific task management
- MongoDB database persistence

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Project Structure
- client/ → React frontend
- server/ → Node.js backend

## How to Run
1. Start backend
   cd server
   npm start

2. Start frontend
   cd client
   npm run dev
